# Grid Labs STL Downloads

This directory contains public STL files and related printable hardware assets.
The website links to `downloads/stl/` as the permanent catalog path.

Published releases:

- `cloudline-mine4heat-adapter/` - three-piece Cloudline Mine4Heat Adapter
  package with STL files, the original Fusion 360 source, renders, and the
  public Autodesk share link.

Planned categories:

- Fan adapters
- Case and duct parts
- Mounts and fixtures

Add finalized `.stl`, `.3mf`, `.step`, or `.stp` files here when the designs are
ready for public release. Put each release in its own folder with a product page,
direct download links, source files when practical, renders, and license or print
notes when available.
