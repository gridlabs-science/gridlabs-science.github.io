# Grid Labs STL Downloads

This directory is reserved for public STL files and related printable hardware
assets. The website links to `downloads/stl/` as the permanent catalog path.

Planned categories:

- Fan adapters
- Case and duct parts
- Mounts and fixtures

Add finalized `.stl`, `.3mf`, `.step`, or `.stp` files here when the designs are
ready for public release, then update `index.html` in this directory with direct
download links and license notes.
